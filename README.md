# Diabetes Prediction using Machine Learning

## Project Overview
This project predicts the onset of diabetes using diagnostic health data.

The project includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Feature correlation analysis
- Machine learning preparation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset Features
- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age
- Outcome

## How to Run

### Install dependencies
pip install -r requirements.txt

### Start Jupyter Notebook
jupyter notebook

## Project Workflow
1. Load Dataset
2. Data Cleaning
3. Exploratory Data Analysis
4. Visualization
5. Correlation Analysis
6. Model Preparation
